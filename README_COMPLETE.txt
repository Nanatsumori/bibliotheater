ビブリオ劇場 完成版（2026年7月）

このZIPは、アップロードされたbibliotheater-githubを基に、次を修正した完成版です。
・完全なstyle.cssをGit履歴から復元
・レパートリーをフライヤー画像入りの囲みデザインに変更
・絵本ドラマリーディングを舞台プログラム風に変更
・iPadでは2列、スマートフォンでは1列になるよう調整
・既存の作品ページ、画像、リンクを保持

使い方：
1. ZIPを展開します。
2. 中のファイルとフォルダを、パソコンのbibliotheaterフォルダへすべて上書きコピーします。
3. GitHub Desktopで変更内容を確認します。
4. Summaryに「トップページ完成版」などを入力し、Commit to main → Push originを押します。

注意：パソコン側の .git フォルダは削除しないでください。
